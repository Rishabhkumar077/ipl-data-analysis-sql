# IPL Data Analysis using SQL

## Objective
I wanted to explore IPL data and understand player and team performance using SQL. The goal was to practice real-world data analysis and derive meaningful insights.

---

## Dataset
The project uses two datasets:
- matches: contains match details like teams and winners  
- deliveries: contains ball-by-ball data (runs, wickets, players)

---

## What I analyzed

- Top run scorers across matches  
- Strike rate of players  
- Players who are consistent across matches  
- Bowlers with most wickets  
- Team performance based on wins  
- Performance across match phases (powerplay, middle overs, death overs)

---

## Key observations

- Death overs contribute a big portion of total runs  
- Some players consistently perform across matches  
- Teams that finish well in death overs tend to win more  

---

## Tools used
- SQL (joins, aggregations, subqueries)
- CSV data

---

## What I learned
This project helped me get better at writing SQL queries and thinking about data from an analysis point of view.
